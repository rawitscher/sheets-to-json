Welcome to the landing page for the Chariot media uploader!

This app can help
entertain distract calm

patients before procedures.

The app is downloaded onto an Android TV compatible device, such as Anker's Nebula Capsule II. The media contained on the app can be updated online, so long as you have access to the media files and Google Sheets. Read on below for assistance with installing, using, and maintaining the Chariot media player app.

App Resources

Media Listlink

A list of all media included in the app. This list can be edited. All three links (sources, card, background) must lead to active, publicly linked files or the content will not display in the app.

Google Play Applink

If you haven't already, download the app from the Google Play store on your Android TV device. You can view the app in the store here.

JSON Datalink

This leads to the API endpoint. The app is built using node.js and extracts data from a Google sheet's very nested original JSON. The final object contains media sorted into an array of key value pairs that correspond to categories and video lists.

Content Storage

Movie/TV Fileslink

This links to a Google Cloud Storage bucket containing all movie files provided for the app. All titles are released under a creative commons license.

Card Imageslink

This links to a Google Cloud Storage bucket containing all card image files provided for the app. All titles are released under a creative commons license.

Background Imageslink

This links to a Google Cloud Storage bucket containing all background image files provided for the app. All titles are released under a creative commons license.
